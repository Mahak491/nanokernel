Nano Kernel Ltd | Bus Scheduling System

Development of integrated planning and bus scheduling system which can be integrated with an Intelligent Transport Management System for public transportation.

Nano Kernel Limited has developed a Bus Scheduling System (Planning / Scheduling solution) for Public Transport. This will enable users to perform automated scheduling using real-time data, dynamically and schedule the services including the operator requirements. The system is a productivity tool that can be used by transport companies in India to improve the efficiency, productivity, and service levels with new dimensions.
